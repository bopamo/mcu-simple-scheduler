Macro Definition Documentation

#define MSS_DEBUG_MODE   (FALSE)

MSS_DEBUG_MODE set to TRUE to activate MSS debug mode. If not used, can be set to FALSE in order to save some memory space

#define MSS_MAX_NUM_OF_MEM   (1)

MSS_MAX_NUM_OF_MEM maximum number of memory blocks used in the MSS application. If MSS_TASK_USE_MEM is set as FALSE, this value will be automativally set to zero

#define MSS_MAX_NUM_OF_MQUE   (1)

MSS_MAX_NUM_OF_MQUE maximum number of message queues used in the MSS application. If MSS_TASK_USE_MQUE is set as FALSE, this value will be automativally set to zero

#define MSS_MAX_NUM_OF_SEMA   (1)

MSS_MAX_NUM_OF_SEMA maximum number of semaphores used in the MSS application. If MSS_TASK_USE_SEMA is set as FALSE, this value will be automativally set to zero

#define MSS_MAX_NUM_OF_TIMER   (1)

MSS_MAX_NUM_OF_TIMER maximum number of timer used in the MSS application. If MSS_TASK_USE_TIMER is set as FALSE, this value will be automativally set to zero

#define MSS_NUM_OF_TASKS   (1)

MAX_NUM_OF_TASKS maximum number of MSS tasks. This shall not exceed the number of bits which the mss_task_bits_t has.

#define MSS_PREEMPTIVE_SCHEDULING   (FALSE)

MSS_PREEMPTIVE_SCHEDULING set to TRUE to activate preemptive scheduling, otherwise the scheduler will work cooperatively.

#define MSS_READY_TASK_BITS_INIT   (0x0001)

MSS_READY_TASK_BITS_INIT initialization value of MSS task ready bits. Each bit set represent the task which is ready to be executed right at the beginning.

#define MSS_TASK_LIST
Value:
{ \
{example_task_1, NULL}, \
}

MSS_TASK_LIST list of task function and parameter pointers of each MSS task. The task function pointer shall be a valid pointer to void (*) (void*) type function while the parameter pointers can take a NULL pointer.

#define MSS_TASK_USE_EVENT   (FALSE)

MSS_TASK_USE_EVENT set to TRUE to activate the MSS event flag module. If it is not used, this option can be set as FALSE to save some memory space.

#define MSS_TASK_USE_MEM   (FALSE)

MSS_TASK_USE_MEM set to TRUE to activate the MSS memory block. If it is not used, this option can be set as FALSE to save some memory space.

#define MSS_TASK_USE_MQUE   (FALSE)

MSS_TASK_USE_MQUE set to TRUE to activate the MSS message queue module. If it is not used, this option can be set as FALSE to save some memory space.

#define MSS_TASK_USE_SEMA   (FALSE)

MSS_TASK_USE_SEMA set to TRUE to activate the MSS semaphore module. If it is not used, this option can be set as FALSE to save some memory space.

#define MSS_TASK_USE_TIMER   (FALSE)

MSS_TASK_USE_TIMER set to TRUE to activate the MSS timer module. If it is not used, this option can be set as FALSE to save some memory space.

typedef uint16_t mss_timer_tick_t

mss_timer_tick_t mss timer tick data type - can be changed according to the application however notice that a timer can be only started with maximum tick value of half of the data type (e.g. 32767 for uint16_t, or 2147483647 for uint32_t)
